# Code of Conduct

This project follows a simple standard: be respectful, constructive, and focused on improving healthcare software safely.

## Expected Behavior

- Use professional and inclusive language.
- Report bugs and security concerns responsibly.
- Keep discussions focused on concrete technical issues.
- Respect users, maintainers, clinicians, and patients affected by the software.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Publishing private data, credentials, or patient information.
- Submitting malicious code or intentionally unsafe healthcare workflows.

## Enforcement

Maintainers may close issues, remove comments, or block contributors who violate this code of conduct.
