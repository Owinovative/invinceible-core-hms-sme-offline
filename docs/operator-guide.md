# Operator Guide

## First Run

1. Launch the desktop app.
2. Enter the facility name, address, phone, first admin name, username, and a password of at least 10 characters.
3. Log in with the admin account.
4. Register patients from the Patients page.

## Daily Workflow

1. Reception registers or finds a patient.
2. Reception creates a visit.
3. Triage captures vitals and chief complaint.
4. The doctor opens the doctor queue and completes the consultation.
5. The doctor orders lab tests or sends prescriptions when needed.
6. Lab records results.
7. Pharmacy marks prescriptions as dispensed.
8. Billing creates invoice lines and records payments.
9. Reports shows daily totals.

## Backup

Use the Backup page to export the SQLite database. Copy backup files to an external drive at the end of each day.

Restore replaces the current database with a selected `.sqlite` backup file. Close active workflows before restoring.
